# 공부시간 측정 프로젝트 (challenge_timer)

- 혼자 공부시간을 측정하고, 상대방과 대결까지 할수 있는 어플리케이션

## 적용 기술

1. React (CRA)
2. TailWindCSS
3. Recoil
4. Firebase
5. Jest/React-testing-library
